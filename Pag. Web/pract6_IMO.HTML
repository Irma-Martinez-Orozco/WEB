"NAVEGADOR"
Es un software, aplicacion o programa que permite el acceso a la web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos.
"BUSCADOR"
Es un programa que te permite ver paginas en internet
"SMTP"
(Es un protocolo para tranferencia simple de correo), Es un protocolo de comunicacion que permite el envio de correos electronicos de correos en internet.
"FTP"
Es un protocolo de red empleado para copiar archivos de una computadora a otra a traves de internet.
"HTTP"
Es un protocolo de red (un conjunto de reglas a seguir) para publicar paginas de internet.
"MetaBuscador"
Es un sistema que localiza informacion en los motores de busqueda mas usados y carese de base de datos propios.
"Red Social"
Son comunidades virtuales donde sus usuarios interactuan como personas de todo el mundo, con quienes se encuentran gustos o intereses en comun.
"TCP/IP"
Es un sistema de protocolo que hacen posibles servicios TELNET, FTP, E-MAIL, y otros entre ordenadores que no pertenecen a la misma red.
"Servidor Web"
Es un programa informatico que procesa una aplicacion de lado del servidor, realizando conexiones bidireccionales o unidireccionales.
"PHP"
Es un lenguaje de codigo abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en html.
"ASP"
Es una tecnologia dinamica funcionando de lado del servidor lo que significa que cuando el usuario solicita un documento asp, las instrucciones de programacion dentro del scrip son ejecutadas para enviar al navegador unicamente el codigo html resultante.
"AJAX"
Se define como una tecnica para el desarrollo de paginas web que implementan aplicaciones interactivas.
"JQUERY
Permite simplificar la manera de interactuar con los documentos html, manipular el arbol DOM, manejar eventos, desarrollar animaciones y agregar interaccion con la tecnica ajax a paginas web.
"JavaScript"
Es un lenguaje con muchas posibilidades, utilizado para crear pequeños programas que luego son insertados en una pagina web y en programas mas grandes, orientado a objetos mucho mas complejos.
"Blog"
Es una pagina de internet que se actualiza peridicamente con material nuevo, que usualmente es publicado por una persona, que expresa pensamientos u opiniones de forma de prosa e incluso algunos llegan a tomar el formato de un diario personal.
"Nube"
Es una metafora empleada para hacer referencia a servicios que se utilizan a traves de internet.
"Streaming"
Es un termino que hace referencia al echo de escuchar musica o ver videos sin necesidad de descargarlos, si no que se hace por fragmentos.
"Router"
Es un hardware que funciona como un semaforo para controlar las señales que trafican dentro de una red. Determina que debe ir a donde y da un nivel mas de seguridad a tu red.
"Ancho de Banda"
Es una medida de recursos disponibles para transmitir datos. Tambien es una medida que se usa para definir la velocidad de internet o, de forma mas precisa, la velocidad de tu conexion a internet.
"VPN"
Se usa para una red de comunicaciones que se usa para un infraestructura de internet para promover conexiones remotas a redes de una organizacion, por ejemplo a empleados que trabajan desde su casa o estan de viaje.
